Windmill

Vetran je mašina koja pretvara energiju vetra u
rotacionu energiju. Istorijski gledano, koristila se za
mlevenje žita, sečenje drva ili pumpanje vode.
Karakteristična krila i visoka struktura su
ikonografski prizor u holandskom pejzažu.
