= 1. Opšti pregled Mule-a

== Šta je MuleSoft i koji problem rješava

MuleSoft nudi jednu od najrasprostranjenijih integracionih platformi za povezivanje SaaS i enterprise aplikacija u cloud i on-premise okruženjima.
Drugim riječima, MuleSoft je platforma koja IT odjelima daje alate za automatsku integraciju sistema i podataka, ubrzavanje tokova posla i kreiranje digitalnih iskustava – sve na jedinstvenoj, jednostavnoj platformi.

U modernim preduzećima postoji mnoštvo različitih sistema (ERP, CRM, baze podataka, cloud servisi, interne i eksterne aplikacije) koji moraju razmjenjivati podatke.
MuleSoft rješava problem povezivanja tih raznolikih sistema obezbjeđujući skalabilnu, sigurnu i održivu integracionu platformu.

== Šta je Mule 4

U središtu MuleSoft platforme nalazi se *Mule 4 runtime* – integracioni engine i razvojna platforma za izgradnju integracionih aplikacija i API-jeva.
Mule 4 je napisan u Javi i zasniva se na *event-driven arhitekturi (EDA)*; to je lak i skalabilan engine koji automatski upravlja izvršavanjem niti i događaja.

Za developera to znači da se fokus može pomjeriti sa low-level detalja (kao što su thread-ovi, socketi i memorija) na dizajniranje tokova integracije (*flow-ova*) i poslovne logike.

Tehnički gledano, Mule runtime koristi provjerene Java tehnologije u pozadini (npr. izgrađen je na Spring okviru), ali programer ne piše klasičnu Java aplikaciju.
Mule pruža viši nivo apstrakcije kroz vizuelno komponovanje integracionih komponenti.
Cilj je da se integracije grade deklarativno i brže, uz minimalno tzv. *„glue“ kôda*.

Mule 4 predstavlja evoluciju Mule 3 runtime-a.
Zadržava iste ključne elemente (aplikacije, flow-ove, konektore, DataWeave, itd.), ali uvodi jednostavniji model poruka i bogatiji set mogućnosti, tako da postoji manje koncepata koje je potrebno naučiti i održavati.

== Razvojni alati i platforma

Zvanično razvojno okruženje za Mule je *Anypoint Studio* – IDE baziran na Eclipse-u, prilagođen za vizuelno dizajniranje Mule aplikacija.

U Anypoint Studio-u:

* kreiramo Mule projekte,
* dodajemo konektore iz bogate palete (preko 200 gotovih konektora za sisteme kao što su Salesforce, SAP, Workday, baze podataka i drugi),
* razvijamo flow-ove drag-and-drop metodom i konfiguracijom komponenti.

Studio dolazi sa svim potrebnim runtime elementima (ugrađeni Mule server, predinstaliran JDK i Maven) i omogućava lokalno testiranje i debagovanje aplikacija.

MuleSoft je uveo i *Anypoint Code Builder* – novu generaciju IDE-a zasnovanu na Visual Studio Code platformi.
Code Builder pruža iste funkcionalnosti kao Studio, ali je dostupan:

* kao VS Code ekstenzija (desktop),
* kao cloud IDE unutar Anypoint Platforme.

To omogućava razvoj Mule aplikacija direktno u pretraživaču uz povezivanje na Anypoint nalog, što donosi dodatnu fleksibilnost, posebno za timski rad i rad na različitim operativnim sistemima.

Pored lokalnog razvoja, MuleSoft obezbjeđuje centralnu platformu za deploy i upravljanje aplikacijama – *Anypoint Platform*.

Unutar platforme, *CloudHub* predstavlja cloud okruženje (MuleSoft-ov iPaaS) gdje se Mule aplikacije mogu deployovati jednim klikom ili skriptovano.

Tipičan razvojni ciklus izgleda ovako:

. razvoj i testiranje lokalno u Anypoint Studio-u,
. deploy aplikacije na CloudHub (sandbox ili production),
. upravljanje aplikacijom kroz Anypoint Platform alate (npr. Runtime Manager).

Studio je direktno integrisan sa Anypoint Platformom, što omogućava deploy aplikacija u cloud direktno iz IDE-a.
Lokalni razvoj i cloud izvršavanje nisu odvojeni procesi, već dio jedinstvenog razvojnog ciklusa.

== MuleSoft zajednica i certifikacija

Za početak rada sa MuleSoft-om formalna certifikacija nije obavezna.
Ipak, *MuleSoft Certified Developer (MCD) Level 1* predstavlja dobar pokazatelj osnovnog znanja koje se očekuje od Mule developera.

Preporučuje se korišćenje domena obuhvaćenih ovim certifikatom kao vodiča kroz gradivo, uključujući:

* rad u Anypoint Studio-u,
* razvoj flow-ova,
* korišćenje konektora,
* osnovne transformacije podataka,
* error handling.

Certifikat služi kao provjera znanja, ali učenje tehnologije se može započeti i bez njega.

Zvanična MuleSoft dokumentacija nudi veliki broj besplatnih resursa:

* Trailhead moduli,
* zvanični tutorijali,
* community forumi,

koji značajno pomažu početnicima u savladavanju Mule 4 platforme.
